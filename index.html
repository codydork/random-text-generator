Comprehensive Guide for Random String Generator

Welcome to the Random String Generator! This tool allows you to create custom random strings based on your specifications. Here's a guide to help you understand how to use all the features and options:

1. Number of Strings

Number of strings to generate: Enter the number of random strings you want to generate in the input box labeled "Number of strings to generate". You can generate up to 1000 strings at a time.

2. Custom Template

Custom template: Use the input box labeled "Custom template" to define the structure of your random strings. You can include various symbols to represent different types of characters:

  • %: Random characters from the default set (a-z, 0-9).
  • ~: Random characters from the custom set defined for ~.
  • \: Random characters from the custom set defined for \.
  • |: Random characters from the custom set defined for |.
  • #: Random characters from the custom set defined for #.
  • .: Random characters from the custom set defined for .
  • ,: Random characters from the custom set defined for ,
  • @: Random characters from the custom set defined for @.
  • ?: Random characters from the custom set defined for ?.
  • +: Random characters from the custom set defined for +.
  • _: Random characters from the custom set defined for _.
  • *: Random characters from the custom set defined for *.
  • ^: Random characters from the custom set defined for ^.
  • &: Random characters from the custom set defined for &.
  • $: Random characters from the custom set defined for $.
  • £: Random characters from the custom set defined for £.
  • ¬: Random characters from the custom set defined for ¬.

You can specify a fixed length by repeating the symbol (e.g., %%% for 3 random characters). To specify a range of lengths, use the format [symbol(min-max)] (e.g., [%](3-5) for 3 to 5 random characters).

3. Custom Characters

For each symbol, you can define custom characters by entering them one per line in the respective text areas. For example, for %, you can enter: 

a
b
c
1
2
3

This will make the generator use only these characters for the % symbol.

4. Delimiter

Delimiter: Enter a character or string to separate each generated random string. For example, if you enter a comma, the output strings will be separated by commas.

5. Generate and Download

Generate: Click the "Generate" button to create the random strings based on your template and options.

Download as .txt: Click the "Download as .txt" button to download the generated strings as a text file.

Examples

Here are a few examples of custom templates:

  • prefix-%(3-5)-suffix: Generates a string with "prefix-", followed by 3 to 5 random characters, and ending with "-suffix".
  • [%(5)]: Generates a string with 5 random characters.
  • [£(2-4)]@[?(3)]: Generates a string with 2 to 4 random £ characters, followed by "@", and then 3 random ? characters.
